# 📝 Todo App

A simple and responsive **Todo App** built using **HTML, CSS, JavaScript, and Bootstrap 5**. This project allows users to add, complete, and delete tasks through a clean and user-friendly interface.

---

## 🚀 Features

* ➕ Add new tasks
* ✅ Mark tasks as completed
* 🗑️ Delete tasks
* ⌨️ Add tasks by pressing the **Enter** key
* 📱 Responsive design using Bootstrap 5
* 🎨 Clean and modern UI

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Bootstrap 5

## ▶️ How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/Muhammad-Arshiyan/todo-app.git
```

2. Open the project folder.

3. Double-click **index.html** or open it with **Live Server** in VS Code.
